# 📑 VPR 발표 자료

---

## 1. 서비스명

**VPR — Virtual Program Runner**

> 부제: *"설치 없이, 브라우저에서 바로 코딩"*

---

## 2. 서비스 소개

> 프로그래밍 도구 설치 없이 웹에서 즉시 코드를 작성하고,  
> AI 도움으로 에러를 해결하는 **교육용 온라인 코딩 플랫폼**

---

## 3. 만든 계기

- 새 언어를 배우려면 설치부터 해야 하는 번거로움
- 오류가 어디서 나는지 찾기 어려운 경험
- 초보자가 코딩을 시작하는 진입 장벽이 너무 높음

---

## 4. 타겟

| 대상 | 설명 |
|------|------|
| 🎓 코딩 입문자 | 처음 프로그래밍을 배우는 학생·비전공자 |
| 💡 빠른 테스터 | 설치 없이 아이디어를 즉시 검증하고 싶은 사람 |
| 💾 저장공간 절약형 | IDE·컴파일러 설치 부담이 있는 사용자 |
| 🚨 에러 해결 필요자 | 오류 원인을 몰라서 막힌 사람 |

---

## 5. 핵심 기능

1. **브라우저 즉시 코딩** — 설치 없이 JS·Python·C/C++·Java·Go·Rust 등 16개+ 언어 코드 작성
2. **AI 코드 도우미** — 아이디어 생성·코드 개선안·단계별 설계 제공 (HuggingFace 연동)
3. **에러 분석 도구** — 오류 메시지 붙여넣기 → 원인 추정 + 수정 방향 제시
4. **HTML 실시간 미리보기** — HTML/CSS 코드 결과를 에디터 옆에서 즉시 확인

---

## 6. 부가 기능

1. **예제 코드 라이브러리** — 언어별 샘플 코드 열람·에디터로 바로 불러오기
2. **계정 & 프로필** — 회원가입·로그인·프로젝트 저장·프로필 편집
3. **Arduino/ESP-IDF 회로 시각화** — 핀 연결 회로도 템플릿 제공
4. **한/영 다국어 UI + 라이트·다크 테마**

> 🚧 커뮤니티 공유 — 구조 설계 완료, 실운영 기능 개발 예정

---

---

## 8. 장점

- ✅ **올인원 구조** — 코딩·에러해결·AI·예제가 한 서비스 안에
- ✅ **에러 룰 분석** — 패턴 기반 오류 원인 즉시 표시 (오프라인 동작)
- ✅ **HTML 미리보기 즉시 동작** — 별도 설정 없이 바로 렌더링
- ✅ **한국어 친화 UX** — 초보 한국어 사용자 중심 설계
- ✅ **경량 스택** — Node.js + 순수 HTML/JS, 빠른 로드

---

## 9. 단점 / 한계

- ⚠️ **코드 실행은 런타임 세팅 필요** — 기본 상태에서 즉시 실행 안 됨
- ⚠️ **AI는 HuggingFace API Key 필요** — 무료이나 발급·설정 과정 존재
- ⚠️ **커뮤니티 공유 미완성** — UI 구조만 있고 실제 공유 기능 개발 예정
- ⚠️ **Arduino는 회로 시각화만** — 실제 빌드·플래시 미지원
- ⚠️ **API Key 브라우저 로컬 저장** — 공개 배포 시 보안 프록시 필요
- ⚠️ **멀티유저 실운영 DB 미완성** — 현재 로컬 SQLite MVP 수준
- ⚠️ **오프라인 실행 불가** — 코드 실행은 인터넷 필수

---

## 10. 기대 효과 & 비전

> **"누구나 5초 안에 코딩을 시작할 수 있는 환경"**

- 코딩 교육 현장에서 IDE 설치 없이 수업 즉시 시작
- 에러로 막힌 초보자가 스스로 원인을 찾는 경험 제공

### 향후 로드맵

| 단계 | 내용 |
|------|------|
| 1단계 | 코드 실행 인프라 안정화 |
| 2단계 | 커뮤니티 공유 기능 완성 |
| 3단계 | Arduino 실제 빌드·플래시 지원 |
| 4단계 | 실서버 DB 전환 및 멀티유저 지원 |

# 향상된 부품 렌더링 계획

## 1. SVG 크기 확대
- 현재: 140x50px
- 새로 변경: 180x100px (더 큰 화면에서 부품이 더 명확함)

## 2. 부품별 렌더링 개선

### MCU 보드
- Arduino: 실제 보드 모양 (검정색 바탕)
- ESP32: 파란색 보드 (ESP 시리즈 구분)
- 핀 표시 (양쪽에 2줄씩)

### LED (RGB 지원)
- 실제 LED 패키지 모양 (원형)
- ON 상태: 밝은 색상 + 빛 효과
- OFF 상태: 어두운 색상

### 디스플레이
- OLED: 검정색 + 노란색 테두리 + 노란 텍스트
- TFT: 파란색 바탕 + 흰색 텍스트
- LCD: 검정색 바탕 + 녹색 텍스트

### 저항 (Resistor)
- 더 사실적인 저항체 모양
- 갈색 몸체 + 색상 띠

### 커패시터 (Capacitor)
- 원형 전해 또는 평면형
- 극성 표시

### 코일/인덕터 (Inductor)
- 나선형 모양

### 수동 부품
- 다이오드: 삼각형 + 칸
- 트랜지스터: 3개 핀 표시
- 변압기: 2개 코일

## 3. 와이어 연결 개선
- 드래그 앤 드롭 지원
- 연결 선 더 굵게 및 색상 구분
- 시각적 피드백 (연결 중일 때 점선)

## 4. 레이아웃 개선
- Breadboard-like 그리드 시스템
- 상자 완전 제거
- 부품들이 직접 보이도록 (라벨 최소화)
